# wb.zk789.cn_spider

四川省自考网上报名系统 爬虫

## 依赖

Python 3+

``` pip install requests lxml prettytable Pillow beautifulsoup4 ```

## 使用

下载并解压项目打包文件

https://github.com/blackyau/wb.zk789.cn_spider/archive/master.zip

直接在当前目录运行 run.bat 即可
